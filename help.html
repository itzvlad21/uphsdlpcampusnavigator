Frequently Asked Questions

How do I navigate the campus map?

You can navigate the campus map using several methods:

  • Touch and drag to look around in the 3D view
  • Pinch to zoom in or out
  • Use the navigation controls to select specific locations
  • Click on map markers to view detailed information about facilities

The map also supports voice-guided navigation to help you reach your destination.

What should I do if the app isn't working?

If you're experiencing issues with the Campus Navigator, try these troubleshooting steps:

  1. Ensure you have a stable internet connection
  2. Clear your browser cache and cookies
  3. Try refreshing the page
  4. Use a different browser (Chrome or Firefox recommended)
  5. Disable any browser extensions that might interfere

If problems persist, please contact our support team using the contact information provided on this page.

Can I use the app as a guest?

Yes, the UPHSD-LP Campus Navigator is accessible to guests without requiring an account. Guest users can:

  • Explore the 3D campus map
  • View information about buildings and facilities
  • Use basic navigation features

However, some advanced features may be limited to registered students and faculty members. You can log in as a guest from the main login page.

Does the app work on mobile devices?

Yes, the UPHSD-LP Campus Navigator is fully responsive and optimized for mobile devices. The application works on smartphones and tablets with these recommended specifications:

  • Modern browser (Chrome, Safari, Firefox)
  • Device with sufficient processing power for 3D rendering
  • Stable internet connection

The mobile experience includes touch controls specifically designed for smaller screens.

How accurate is the navigation system?

The UPHSD-LP Campus Navigator provides highly accurate directions based on the latest campus layout data. The system:

  • Offers detailed path information between buildings
  • Updates regularly to reflect campus changes
  • Includes interior navigation for supported buildings

While the system strives for maximum accuracy, occasional discrepancies might occur due to recent campus changes. We regularly update the system to ensure the most current information.
