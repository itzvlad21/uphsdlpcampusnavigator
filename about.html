UPHSD-LP Campus Navigator

An Interactive 3D Navigation System

Project Overview

The UPHSD-LP Campus Navigator is a web-based interactive navigation system designed specifically for the University of Perpetual Help System DALTA Las Piñas Campus. This innovative tool addresses navigation challenges faced by students, faculty, and visitors through real-time directions, an interactive 3D campus map, and comprehensive facility information.

Using modern web technologies including Node.js and Express.js for backend processing, the system integrates with real-time navigation tools and database management systems to provide a seamless experience.

Research Context

Navigation within large university campuses poses significant challenges for students, faculty, and visitors. The University of Perpetual Help System DALTA Las Piñas Campus, with its expansive layout and numerous facilities, faced several wayfinding challenges:

  • Inefficient navigation using static maps and physical signage
  • Limited access to real-time campus information
  • Difficulties in locating specific buildings and offices
  • Lack of integrated facility information

Research Questions

  1. How can a web-based interactive map enhance campus navigation efficiency?
  2. What impact does real-time navigation have on user experience?
  3. How effectively can digital tools integrate with physical campus infrastructure?
